redis基本使用

Hexo
2020-05-13

redis安装：

​ Ubuntu版：

​ wget http://download.redis.io/releases/redis-3.2.8.tar.gz

​ 解压：tar -zxvf trdis.3.2.8.tar.gz

​ 复制到/usr/local/redis下，sudo mv ./redis-3.2.8 /usr/local/redis

​ 需要有c语言环境 sudo apt-get install gcc

​ sudo apt-get install make

​ sudo make

​ sudo make test //测试

​ 有可能出现依赖没有的错误：

​ wget http://downloads.sourceforge.net/tcl/tcl8.6.1-src.tar.gz

​ sudo tar -zxvf tcl8.6.1-src.tar.gz -C /usr/local/tcl8.6.1

​ cd /usr/local/tcl8.6.1/unix/

​ sudo ./configure

​ sudo make

​ sudo make install

​ 安装到/usr/local/bin/目录:sudo make install

​ 使用脚本安装redis ./utils/install_server.sh（在里面个性化设计自己的配置）

redis的开启与关闭：

服务端：

开启：sudo redis-server /etc/redis/redis.conf(6379.conf)

重启：sudo service redis restart

关闭：sudo kill -9 redis-pid

客户端：

redis-cil

redis排错：

1.进入log目录，打开log文件，shift+g查找错误信息。

2.直接删除Data dir下面的东西。

redis数据类型：

string：

select 数据库number（默认十六个数据库，就是[0-15]） 选择要操作的数据库

增：set key value

查：get key value

改：对已有数据库的操作就是改数据

删：del

hash类型：

设置：hset

获取：hkeys（左侧一列）,hvals（右侧一列）

删除：del,hdel

查找：hmget

list类型：

增：lpush,rpush 从左插入和从右插入

删：lrem key count value //删除指定数量的value del可直接对list对象删除

改：更新

查：lrange key index index //一个索引，到一个索引 -1表示最后一个位置

插入：linsert key before|after value insert-value //在一个值前或后插入一个值

弹出：rpop 删除最后一个元素，返回值为弹出的元素。

set操作：

自动去重

增：sadd key member…

删：srem key member… ，spop弹出最后一个元素

改：

查：smembers key，scard key(返回集合中元素个数)

移动：smove resource target item 是移动，不是复制。

zset类型：

增：zadd

删：zrem

改：

查：zrange(通过score的区间来查看)

​ zrangebyscore(通过key排序，来取value)

​ zscore (查看集合中元素个数)

​ zscore(查看指定value的key)
