vim简单使用

Hexo
2020-05-13

vim用法：

查找模式：

/string 向下查找 ?string 向上查找 #查找光标覆盖的单词

替换：末行模式 r

位置与移动：

G：文件末 gg：文件头 行首：^ 行末：$

e:每段string移动

ctrl+o 回到之前那个位置 ctrl+i回到之前回到的那个位置

复制剪切等：

复制： yy nyy(n行复制)

粘贴：p P

撤销：u 反撤销：ctrl + r

缩进：>>向右 <<向左

查找man文档：shift + k

分屏：

屏幕分屏：vsp垂直分屏 sp水平分屏 sp/vsp + 文件

分屏下的光标移动：ctrl +w +(hjkl) 上下左右移

可视模式：v 或 V 或ctrl+v

可视块文本：ctrl+v

可选中文本块，选中文本块，执行所有可以执行的操作。

应用：

1.在所有行前插入指定内容：进入可视块模式，G选到最后一行，shift+i进入插入模式，写入内容，esc以后，即可全部插入

2.在选中的每行前加一样的string：选中，：进入末行模式，：normal 要添加的string normal A.png A代表行末。

程序员：

引入外部命令的结果：r !命令 例：r !ls

修改配置文件：

绪论：在家目录下新建一个文件夹.vim 进入新建一个vimrc的文件

设置快捷键：

noremap更改键位的命令

map

operation与motion：

operation是一个动作，motion是一个意图

比如：d是一个删除动作，->与<-是意图，d + ->就表示向右删一个空格

​ d + d就表示删除这一行

​ y表示复制，y + 3-> 表示向右复制三个字符

​ ci” c->change i “删除“ ”内的内容

​ df: d 删除 f for : 冒号 从当前位置删除到：位置

​ f: f found查找， ：冒号 查找本行的：

下载插件：

实例：

​ 当vim编辑文件，想要保存时，权限不够：

  1. w ~/Desktop/toilet.conf 保存到其他文件，再复制到此处
  2. w !sudo tee % %表示当前文件 输入密码，即可保存

打开文件：gf：go to file 进入光标所在的那个文件，ctrl+o就可返回上一个文件
